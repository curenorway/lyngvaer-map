# Lyngvær Map

An interactive map showcasing Lyngvær, a fishing village in Lofoten, Norway.

## Features

- **Scroll-triggered map animation** - Fly through Norway from overview to Lyngvær as you scroll
- **Interactive location markers** - Clickable pins showing points of interest
- **Info boxes** - Display location images and drive times
- **Smooth animations** - Markers render directly on the map canvas for perfect sync
- **Custom map styling** - Uses custom Mapbox style for authentic Norwegian terrain

## Technology

- Mapbox GL JS v3.0.1
- Custom map style
- Native Mapbox symbol layers for optimal performance

## Deployment

The map is hosted on GitHub at: https://github.com/curenorway/lyngvaer-map

To deploy via GitHub Pages:
1. Go to repository Settings
2. Navigate to Pages
3. Select main branch as source
4. Your site will be live at: https://curenorway.github.io/lyngvaer-map

## Local Development

Simply open `index.html` in a web browser.
